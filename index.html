Who am I?
Kirsten Swanson A Front-End Developer A Creator An Adventurer

Style Guides

Created style guides for a navigation bar, a footer, an accordion tab, and content cards. Each element includes media queries for responsive design.

Tools: HTML | Sass | jQuery

Pomodoro Application

A pomodoro timer that has a default start time of 25 minutes with a 5 minute break time. The user can increase or decrease both the session length and break length. A sound will alert the user when the session and break are over.

Tools: HTML | Sass | JS | ES6

Chat Application

An application that allows the user to login to Firebase and have their messages stored in it. Other users can login as well to chat and the messages will be saved based on the user's name and email.

Tools: HTML | Sass | React | Firebase

Let me introduce myself

Hi, I'm Kirsten! I am an adventurous, risk-taker who seeks a creative challenge. My journey into the ever-changing tech world started with my passion to use my creativity in a versatile yet impactful way. It was actually through my neighbors that programming was introduced to me. They inspired me to pursue this path and to switch careers. My goal is to create memorable and functional user interfaces that are accessible to everyone.

My philosophies on life:

Enjoy life | Dream big | Create long-lasting relationships
Never stop learning | Make mistakes | Travel | Experiment
